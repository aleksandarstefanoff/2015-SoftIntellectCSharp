примитивен тип:
	bite, short, int, long (целочислени)
	float, doubele, decimal (реални, с плаваща запетая)
	bool (boolean) булев тип
	char (символен тип)	
	??object, 
	string - низ от символи

примитивните типове се задават с техните стойност (в сорс код) литерали
например:
int i = 1, примитивният тип int с име i е със зададена стойност (литерал) 1
bool result  = true, примитивният тип bool с име result има стойност (литерал) true
и т.н.


	За да присвоим стойност на тип данни използваме (Console.ReadLine()),
което прочита стойността, която записваме и я присвоява на типа данни
който сме предвидили за това:

	int A = int.Parse(Console.ReadLine());

	Console.ReadLine() прочита това, което е написано в конзолата
	и присвоява стойността му на примитивния тип (в случая int с име А)
	(ако типът не е string трябва да има Parse) 

	За тип string нещата изглеждата така:

	string a = (Console.ReadLine());
